  5. 要素不足

  現在の familyai.jp に存在せず、casual 世界観に必要な部品：

  - polaroid 写真フレーム: 白枠＋回転（+3°, -5°, +4°, -6° 交互）＋温かみシャドウ。ツールカードのアイキャッチや家族写真用
  - マスキングテープ: polaroid の上端に半透明テープ（デフォルト or pink）。1-2枚に絞って使う
  - num-stamp: 14vmin の巨大テラコッタアウトライン数字＋mono 小ラベル。セクション番号やツール番号に
  - corner-stamp: 右上の郵便消印風ラベル「CH.01 · TOOLS」など
  - mustard ハイライト（.hl）: キーフレーズにマスタードの半透明下線。「家族をもっと幸せに」など中核コピーに
  - hand 手書き sign-off: フッター直前の「ありがとうございました」を Caveat 手書き体＋mustard ✦ で
  - ticket motif: ヒーローの「VOL.01」表示を boarding pass 風チケットに変換可能
  - kicker ラベル: mono 体＋テラコッタ＋左ダッシュ線。「AI = 愛」の上の小見出しに

  6. ユーザーへの確認事項

  - 家族写真・ビジュアル素材の有無: casual の中核は polaroid 写真。ツール画面・記事アイキャッチ・家族像など、回転フレームに入れる素材はあるか
  - 伝えたい中心物語: 「AI = 愛」の背景にある創業者の動機や具体的な家族エピソードはあるか。casual は「目睹した具体性」が命なので、抽象的なままでは tone contamination に陥る
  - CTA の優先度: 現状は「ツールを開く」「記事を読む」の2択。casual 化後も同じか、または別のアクション（メルマガ登録・お問い合わせ）を追加するか
  - フォントの日本語表示確認: しっぽり明朝・Zen Kaku Gothic Antique は Google Fonts 提供。本番サイトでのレンダリングを確認済みか

  - casual は印刷・スライド向けで Web LP テンプレートを持たない: report.html が最も縦スクロール Web に近いが、ナビゲーション固定・マルチセクション間の遷移など、Web
