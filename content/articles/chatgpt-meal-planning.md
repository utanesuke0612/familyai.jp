## 毎日の「今日何作る？」問題を10秒で解決

夕方になると悩む献立。冷蔵庫を見ながら「何が作れるかな…」と考える時間を、ChatGPTがたった10秒で解決してくれます。

## ChatGPTへの献立相談の仕方

**コピーして使える！基本プロンプト：**

```
冷蔵庫に以下の食材があります。
夕食の献立を3つ提案してください。

食材：鶏もも肉、玉ねぎ、にんじん、じゃがいも、
      卵、豆腐、ほうれん草

条件：
- 子ども（7歳）も食べられるもの
- 調理時間30分以内
- 主菜と副菜の組み合わせで
```

## 1週間分の献立を一気に作る

```
来週1週間（月〜日）の夕食献立を考えてください。
家族構成：大人2人、子ども1人（7歳）
予算：1週間で約5,000円
栄養バランスを考慮して、買い物リストも一緒に作ってください。
```

## アレルギー対応も簡単

「○○アレルギーがあります」と伝えるだけで、その食材を使わないレシピだけを提案してくれます。

## レシピの詳細も聞ける

献立が決まったら「親子丼の作り方を詳しく教えて」と続けて聞けばレシピも完成。

## まとめ

AIを献立の「相談相手」として使いましょう。毎日の料理のストレスが、ぐっと減りますよ。
