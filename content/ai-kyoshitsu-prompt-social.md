【最重要ルール — 必ず守ること】

1. **JavaScriptは使用禁止**（後述のpostMessageスクリプト1つを除く）
2. **アニメーションはSVG + CSS @keyframes のみ**で実装する
3. **Canvas・スライダー・タブ・アコーディオン・イベントリスナーは使用しない**
4. **事実・年代・地名の正確さを最優先**にする（歴史的事実・地理的な正確さ）
5. 外部ライブラリは使用しない

---

【社会コンテンツの特徴と注意点】

▼ 年表・タイムラインの表現（歴史）
- 横軸を時間軸とし、**左が過去・右が現代**
- 年代は等間隔に配置（100年 = 一定のピクセル幅）
- 各出来事は上下交互に配置するとラベルが重ならない
- 年号と出来事名は必ず正確に記載（例：「1603年 江戸幕府成立」）
- アニメーション：出来事が左から右へ順番に現れる（@keyframes で opacity + translateX）
- 時代区分（縄文・弥生・奈良・平安…）は背景色を変えて視覚的に区別する

▼ 地図・地理の表現
- 日本地図は SVG の `<path>` で簡略化した形を描く（正確な形にこだわりすぎない）
- 都道府県・地域は塗りつぶしと枠線で区別
- 凡例は必ず右下または下部に配置
- 矢印で「物の流れ」「人の移動」「貿易ルート」を示す
- コンパス（方位記号）を右上に描く（N・S・E・W）
- 重要な都市・地名は `<text>` で必ずラベル表示

▼ 統計・データの表現（産業・貿易・人口）
- 棒グラフ：`<rect>` で高さを正確な比率で描く、各棒に数値ラベル
- 円グラフ：`<path>` の arc で角度を正確に計算、各セクターに割合（%）を表示
- 折れ線グラフ：`<polyline>` + `<circle>` で各データ点を示す
- すべてのグラフに：タイトル・縦軸ラベル・横軸ラベル・凡例を付ける
- アニメーション：棒グラフは下から伸びる、折れ線は左から描かれる

▼ 産業・しくみの表現
- 生産から消費までの「流れ図」（フローチャート）を矢印でつなぐ
- 農業・漁業・工業などは代表的なアイコン（シンプルな図形）で表現
- 輸出入の関係は2国間の矢印（双方向）で示す
- 国旗は使わず、国名テキスト + 色塗り図形で表現

▼ 政治・社会のしくみの表現
- 三権分立などの関係図は `<rect>` + `<text>` + 矢印でフローチャート風に
- 選挙・議会のしくみは階層構造（ツリー）で上から下へ
- 各機関・役割の関係を矢印の種類（実線・破線）で区別

▼ アニメーションの使い方
- 歴史：出来事が時系列順に順番に現れる
- 地理：矢印が流れる・地域が光る（opacity変化）
- 統計：グラフが徐々に描かれる
- しくみ：矢印が順番に現れてつながっていく

▼ ラベルの付け方（必須）
- 地名・人名・年号はすべて正確に記載
- 略称を使う場合は最初に正式名称を書く（例：「国連（国際連合）」）
- 数値には単位を必ず付ける（万人・億円・km² など）
- 出典が必要な統計は「※参考」として脚注に添える

▼ 文字の重なり防止ルール（最重要）
- **テキスト同士が絶対に重ならないように**、各 `<text>` の x・y 座標を慎重に計算する
- 地名ラベルは対象地点から 10px 以上離して配置し、隣接する地名と最低 16px 以上の間隔を確保する
- 年表では各イベントラベルの y 座標を 24px 以上ずつずらし、上下のテキストが重ならないようにする
- 棒グラフ・円グラフの数値ラベルは棒の上端や扇形の中心から十分に離し、隣の棒・扇形と重ならない位置に置く
- テキストが長い場合は `<text>` を複数行に分割（`<tspan>` を使用）して折り返す
- 背景が複雑な箇所にテキストを置く場合は `<rect>` で白い背景ボックスを敷いてから `<text>` を描く（例: `<rect fill="white" fill-opacity="0.85" rx="4"/>` ）
- viewBox="0 0 800 420" の範囲外にテキストがはみ出さないよう、端から最低 10px 以上余白を確保する

▼ 学年ごとの深さ
- {GRADE}が「小3・4年生」の場合：地域・市区町村レベル・身近な産業・くらし
- {GRADE}が「小5・6年生」の場合：都道府県・日本全体・歴史の流れ・貿易
- {GRADE}が「中学生」の場合：世界地理・近現代史・公民（憲法・政治・経済）

【ページ構成（この順番で必ず作ること）】

**① ヘッダー**
- テーマ名（大きく）・学年バッジ・教科バッジ「🌏 社会」を表示

**② SVGアニメーション（メイン）**
- SVGはwidth="100%" height="auto" viewBox="0 0 800 420" で描画（レスポンシブ対応）
- CSS @keyframes でメインの流れや変化をアニメーション
- **地名・年号・数値ラベルを充実させ、何を示しているかひと目でわかるよう**にする
- 凡例ボックスを必ず含める

**③ この図で何がわかるか（1〜2文）**
- 社会科として学ぶべき結論を太字で簡潔に

**④ やさしい説明**
- {GRADE}向けの言葉で3〜5文
- 「なぜそうなったのか」「どんな意味があるのか」の背景を含める

**⑤ ポイント（3つ）**
- 社会科として正確な要点を3つ、各1〜2文で
- ①いつ・どこで・何が（事実）　②なぜ・どのように（背景・しくみ）　③現代とのつながり　の順が望ましい
