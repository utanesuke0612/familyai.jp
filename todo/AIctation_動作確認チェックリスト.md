## 🚨 トラブルシューティング

### Q: 「⚠️ 音声を読み込めませんでした」が出る
**A**: 以下を順に確認：
1. `.env.local` の `NEXT_PUBLIC_VOA_BLOB_BASE` が**Public Blob ホスト**（`.public.blob.vercel-storage.com`）になっているか
2. dev サーバを再起動したか（env 変更は再起動必須）
3. URL 先頭が `https://` か（`http://` だと CORS）
4. MP3 が Vercel Blob に存在するか（パス確認）

### Q: lesson-01 のセンテンスリストが空
**A**: 
1. `content/voaenglish/01_01_Anna/lesson-01.sentences.json` が存在するか
2. JSON のパース成功しているか（DevTools Console に warning 出てないか）
3. `pnpm db:convert-srt content/voaenglish/01_01_Anna/lesson-01.srt` を再実行

### Q: 自己申告 → 進捗が DB に保存されない
**A**:
1. ログイン済みか（未ログインなら localStorage）
2. `lessons_progress` テーブルが本番 DB に存在するか（migration 0013）
3. DevTools Network で `POST /api/user/lessons-progress` のステータスコード確認
4. 401 → セッション切れ・再ログイン
5. 403 → CSRF 失敗（origin 不一致）
6. 500 → サーバ側の DB エラー → Vercel ログ確認

### Q: 🌟 押しても confetti が出ない
**A**:
1. ブラウザコンソールに `[DictationPanel] confetti failed` warning が出てないか
2. `canvas-confetti` がインストール済みか（`pnpm list canvas-confetti`）
3. 動的 import が成功しているか（Network タブで `canvas-confetti.js` 読み込み確認）

### Q: モーダルが Esc で閉じない
**A**: input/textarea にフォーカスがあると無視される設計。input から Tab キーで一旦外してから Esc を押す。
