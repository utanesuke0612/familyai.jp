# AI Echo — 設計書
# 用途: CodingAgentへの実装指示書
# 対象: AIctationページ内のアウトプット評価機能
# バージョン: 1.0

---

## 概要

```
機能名: AI Echo
配置:   AIctationページの聴写プレイヤーの下
        （デフォルトで常に表示）
目的:   聴写で学んだ内容を自分の言葉でアウトプットし
        AIが温かく評価・フィードバックする

「聴いて・書いて・自分の言葉で表現する」
この3ステップで学習の定着率を高める。
```

---

## 表示タイミング

```
⚠️ 重要:
  聴写完了後に表示するのではなく
  ページ読み込み時からデフォルトで表示する。

  → ユーザーがいつでも好きなタイミングで使える
  → 聴写前の予習アウトプットにも使える
  → 聴写後の復習にも使える
```

## タブの設計

```
3つのタブを横並びで表示（均等幅）

スタイル:
  選択中: #FF8C42（オレンジ）背景・白文字・bold
  未選択: #FDF6ED（クリーム）背景・#B5896A色
  境界線: #E8CFA8
  border-radius: 上部のみ rounded

デフォルト選択: 🌱 Level 1

タブ切り替え:
  クリックでコンテンツエリアを即切り替え
  入力中のテキストはタブ切り替え時にリセット
  （確認ダイアログは不要・シンプルに）
```

## フィードバックの表示

```
送信ボタンを押すと
入力欄の下にフィードバックが表示される

┌─────────────────────────────────────┐
│ 🤖 AI Echo からのフィードバック      │
│                                     │
│ ✅ 良かった点:                      │
│ （褒めるコメント・1〜2文）           │
│                                     │
│ 💡 一つだけ提案:                    │
│ （改善点 + 具体的な言い換え例）      │
│                                     │
│ 🌱/🌿/🌳 （励ましの言葉）           │
│                                     │
│ [もう一度書く]                      │
└─────────────────────────────────────┘

ローディング中:
  「🤖 AI Echo が考えています...」
  3点ドットアニメーション
```

---

## フィードバックの温度感ルール（必須）

```
必ず守るルール:
  ① 最初に必ず良い点を褒める（1〜2文）
  ② 改善点は1つだけ（多すぎない）
  ③ 改善点は具体的な言い換え例を示す
  ④ 最後は励ましの言葉で締める
  ⑤ 批判的・否定的な言葉は絶対に使わない

NGパターン（絶対禁止）:
  ❌ 「文法が間違っています」
  ❌ 「内容が不正確です」
  ❌ 「もっと詳しく書いてください」

OKパターン:
  ✅ 「ポイントをしっかり押さえています！」
  ✅ 「一つだけ提案すると〜」
  ✅ 「この調子で続けましょう！」
```

## レベル別フィードバック例

---

### 🌱 Level 1 のフィードバック例

```
ユーザーの入力:
  "The president go to Japan.
   They talk about trade."

AIフィードバック:
  ✅ 良かった点:
  大事なポイントをしっかりつかんでいます！
  内容はバッチリです。

  💡 一つだけ提案:
  "go" → "went"、"talk" → "talked"
  過去の話なので過去形を使いましょう。
  でも内容の理解は完璧です！

  🌱 どんどん挑戦してみてください！
  次はLevel 2にも挑戦してみましょう。
```

---

### 🌿 Level 2 のフィードバック例

```
ユーザーの入力:
  "The US president visited Japan
   to discuss trade issues.
   They reached some agreements."

AIフィードバック:
  ✅ 良かった点:
  とても自然な英語です！
  "reached agreements"という
  表現も素晴らしいです。

  💡 一つだけ提案:
  「どんな合意をしたか」を
  一文加えるとさらに完璧になります。
  例: "They agreed to reduce tariffs."

  🌿 Level 2をクリアできています！
  自信を持ってください。
```

---

### 🌳 Level 3 のフィードバック例

```
ユーザーの入力:
  "I think this trade deal is important
   for both countries because
   it will help economy."

AIフィードバック:
  ✅ 良かった点:
  自分の意見と理由を
  しっかり英語で表現できています！
  構成がとても明確です。

  💡 一つだけ提案:
  "help economy" → "boost the economy"
  冠詞のtheを入れると
  より自然なビジネス英語になります。

  🌳 意見を英語で表現できることは
  本当に素晴らしいです！
  この調子で続けましょう。
```

## システムプロンプト（サーバー側）

```typescript
// app/api/ai/route.ts に実装

const getAiEchoPrompt = (
  level: 1 | 2 | 3,
  script: string
) => `
あなたは英語学習の温かいコーチです。
以下のVOAスクリプトを参照して、
ユーザーのアウトプットを評価してください。

【VOAスクリプト】
${script}

【フィードバックのルール（必ず守ること）】
① 最初に必ず良い点を褒める（1〜2文）
② 改善点は1つだけに絞る（多すぎない）
③ 改善点は具体的な言い換え例を示す
④ 最後は励ましの言葉で締める
⑤ 批判的・否定的な言葉は使わない
⑥ 日本語で回答する

${level === 1 ? `
【Level 1の評価観点】
・内容の正確さのみを見る
・文法ミスは厳しく指摘しない
  （内容が合っていれば温かく見逃す）
・3文でなくても減点しない
` : ''}

${level === 2 ? `
【Level 2の評価観点】
・内容の正確さ
・文法の正しさ
・語彙の適切さ
・「誰が・何を・なぜ」が含まれているか
` : ''}

${level === 3 ? `
【Level 3の評価観点】
・スクリプトを正しく理解しているか
・意見が明確か
・理由・根拠があるか
・文法・語彙の質
` : ''}

【評価フォーマット（必ずこの形式で）】
✅ 良かった点: （褒める・1〜2文）
💡 一つだけ提案: （改善点 + 言い換え例）
${level === 1 ? '🌱' : level === 2 ? '🌿' : '🌳'} （励ましの言葉・1文）
`;
```
